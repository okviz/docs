Synoptic Designer is designed as a local browser-based editor. The SVG maps, JSVG projects, bitmap images, and data lists that you create or import in Synoptic Designer are processed locally in your browser.

## No Project Data Sent to OKVIZ

Synoptic Designer does not send your map content, imported drawings, tracing images, imported data lists, browser-local projects, or exported JSVG files to OKVIZ servers or other external servers.

In particular, OKVIZ does not receive:

- SVG or JSVG files that you open or export;
- bitmap images used as tracing references;
- data lists imported to review area binding;
- browser-local project snapshots;
- Synoptic Panel area IDs, titles, binding metadata, or formatting changes created in the editor.

## No Personal Data Sent From Projects

Synoptic Designer does not intentionally collect or transmit personal data contained in files, drawings, labels, imported data lists, project names, author fields, or exported maps.

You are responsible for the content you choose to import or type into a project. If a map, image, file name, author value, or imported data list contains personal data, that content remains local to your browser unless you choose to export, download, share, upload, or otherwise transfer it outside Synoptic Designer.

## Local Browser Storage

When browser-local save is used, Synoptic Designer stores project data in the current browser profile by using browser storage. This storage is local to the device, browser, and profile where the project was saved.

OKVIZ does not operate a cloud backup for Synoptic Designer projects. Clearing browser data, using private browsing, switching profiles, browser storage limits, device loss, or organization policies can remove or block browser-local projects.

## Network Requests

Synoptic Designer may require normal website requests to load the application, documentation, scripts, styles, and static assets from the website that hosts it. Those website-level requests are separate from project content processing.

This policy applies to content created, imported, edited, saved, or exported inside Synoptic Designer. Website access, cookies, analytics, account activity, support requests, purchases, and other OKVIZ website interactions are covered by the applicable OKVIZ website policies.

## Export and Sharing

Exporting a JSVG file downloads a local file from the browser. After export, you control where that file is stored, shared, uploaded, imported, backed up, or published.

If you import an exported map into Synoptic Panel, the data and privacy behavior of Synoptic Panel is governed by the Synoptic Panel documentation and policies.
