This page contains a quick summary of versions and notable changes of Synoptic Designer.

## v2.0.0

**Released:** 2026-07-04

**New Features:**

- Release v2.0 as the best browser-based companion editor for Synoptic Panel maps.
- Add support for creating new maps from JSVG files.
- Add browser-local project saving and loading.
- Add JSVG export with Synoptic Panel map metadata.
- Add an ***Areas*** panel for area IDs, binding review, explicit binding, titles, and ***Do Not Bind*** state.
- Add an import workflow for data lists used to review area matching before export.
- Add tracing image support with opacity, visibility, positioning, and quality controls.
- Add toolbar tools for selection, node editing, panning, drawing, shapes, freehand paths, splitting, images, SVG insertion, grids, text, erasing, and node erasing.
- Add formatting controls for geometry, fill, stroke, opacity, and selection color replacement.
- Add undo, redo, zoom, canvas clearing, and object ordering commands.
- Add contextual help, visual guides, shortcuts, and gesture support.

**Improvements:**

- Overhaul the user interface for a more modern and responsive experience.
- Improve Magic Wand tracing with better edge detection, tolerance, and selection refinement.
