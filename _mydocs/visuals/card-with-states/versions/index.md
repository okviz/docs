Card with States is available in two different versions:

- **Card with States v2**: The current version of the visual. It has been rebuilt on the latest Power BI Custom Visuals APIs and supports the modern Power BI feature set. It also includes the advanced Color Rules manager used across other OKVIZ premium visuals. Card with States v2 is free and does not require a license.

- **Card with States Legacy (v1.x)**: The previous generation of the visual, based on older Power BI APIs. It remains available for a limited time on AppSource to support existing reports, but it will no longer receive updates. This version included legacy limitations and could require a license to unlock some features.

> Card with States v2 uses **Power BI Custom Visuals API v5.11.0**. See [Power BI Requirements](../../requirements.md) for the current platform requirements.

## Versions Comparison

|Version|Status|Power BI API|Licensing|Color Rules|Compatibility|
|---|---|---|---|---|---|
|Card with States v2|Current|v5.11.0|Free, no license required|Advanced Color Rules manager|Not fully compatible with Legacy|
|Card with States Legacy (v1.x)|Legacy, temporary availability on AppSource|Older APIs|May require a license to unlock some features|Legacy rules and options|Cannot be directly replaced by v2|

The two versions are **not fully compatible**, especially for **Color Rules**. For migration guidance, see [Legacy (v1.x)](./legacy.md).
