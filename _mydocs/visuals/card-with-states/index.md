**Card with States** is a Power BI custom visual that displays one or more KPI cards with conditional rules, optional targets, and optional trendlines.

Card with States is available in two versions:

- **Card with States v2**: The current and recommended version of the visual, built on the latest Power BI APIs. This version is free and does not require a license.

- **Card with States Legacy (v1.x)**: The previous generation of the visual, still available for a limited time on AppSource to support older reports. This version will not receive further updates.

Learn more on the [Versions page](versions/index.md).

>> **IMPORTANT**: The documentation on this site refers to the latest version of Card with States (v2) unless otherwise specified. If you are still using the previous version, see [Legacy (v1.x)](./versions/legacy.md) before planning your migration.
