This section contains the options for messages and icons shown by Color Rules.

These settings apply only when a Color Rule defines a message or an icon.

Learn more: [OKVIZ Color Rules](../../../features/color-rules.md)

## Message Text Size

**Default value:** 10

This option defines the text size of rule messages. The value ranges from 5 to 100.

It only applies when a Color Rule defines a message.

## Messages Font Family

**Default value:** `'Segoe UI', wf_segoe-ui_normal, helvetica, arial, sans-serif`

This option sets the font family of rule messages.

## Messages Margin Top

**Default value:** 0

This option adds top spacing above the message area. The value ranges from 0 to 1000.

## Icon Size

**Default value:** 10

This option defines the size of the icon shown by a rule. The value ranges from 5 to 100.

It only applies when a Color Rule defines an icon.

## Reserve Space

**Default value:** Off

This option reserves space for rule messages and icons even when they are not currently shown.

This helps preserve the card layout and prevents content from shifting when messages appear or disappear.
