This section contains the options related to the Color Rules editor in Card with States.

The editor is available in edit mode and lets you define rules that can affect the card background, the data label, the category label, the reference label, the trendline, the trendline area, and optional icons or messages.

Learn more: [OKVIZ Color Rules](../../../features/color-rules.md)

## Show Editor

**Default value:** Off

This option opens the Color Rules editor.

It is only meaningful in edit mode.

## Pin Editor Button

**Default value:** Off

This option keeps the Color Rules button visible on the visual instead of showing it only on hover.

It is only relevant in edit mode.
