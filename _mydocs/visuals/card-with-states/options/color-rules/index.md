This section contains the options related to the Color Rules editor in Card with States.

The editor is available in edit mode and lets you define rules that can affect the card background, the data label, the category label, the reference label, the trendline, the trendline area, and optional icons or messages.

Learn more: [OKVIZ Color Rules](../../../features/color-rules.md)

## Open Color Rules Manager

**Default value:** Off

This option opens the Color Rules manager.

It is only meaningful in edit mode.

## Pin Color Rules Button

**Default value:** Off

This option keeps the Color Rules button visible on the visual instead of showing it only on hover.

It is only relevant in edit mode.

## Message Font

**Default value:** Size 10, default text font, regular style

This composite control defines the font family, size, bold, italic, and underline settings used by rule messages.

These settings apply only when a rule includes a message.

## Icon Size

**Default value:** 10

This option defines the size of the icon shown by a rule.

These settings apply only when a rule includes an icon.

## Margin Top

**Default value:** 0

This option adds top spacing above the message and icon area.

These settings apply only when a rule includes a message or an icon.

## Reserve Space

**Default value:** Off

This option reserves space for rule messages and icons even when they are not currently shown.

This helps preserve layout consistency and prevents content from shifting when rules appear or disappear.
