This section contains basic information and diagnostic options for Card with States.

## Name

The name of the visual.

## Version

The current version of the visual.

## Show Diagnostics

**Default value:** Off

When enabled, the visual writes diagnostic information to the browser console.

This option is mainly useful while authoring or debugging the report and is effective in edit mode.

See also: [Support](../../../issues/support.md#diagnostic-data)

## Inline Console

**Default value:** Off

When enabled, the visual also shows diagnostic information inside the visual area.

This option is available only when **Show Diagnostics** is enabled.

In Power BI Desktop, diagnostic output is also shown inline when diagnostics are enabled.
