Card with States accepts one required measure plus optional fields for targets, color rules, trendlines, and tooltips.

The supported bindings depend on the combination of fields you use:

- **Values** is required.
- If you bind **Category**, both **Values** and **Targets** accept a single measure.
- If you do not bind **Category**, you can bind multiple measures to **Values**. In this case, **Targets** can also accept multiple measures, using the same order as **Values**.
- **Trendline axis** enables the trendline only when each card receives multiple data points.
