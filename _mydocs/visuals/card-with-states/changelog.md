This page contains a quick summary of versions and notable changes of Card with States.  
Card with States uses **Power BI Custom Visuals API v5.11.0**, which requires:
- Power BI Service
- **Power BI Desktop July 2024** or later
- **Power BI Report Server September 2025** or later

[See Power BI Requirements](../requirements.md) for more information.

## v2.0.0

**Released:** 2026-04-03

This release introduces the new generation of Card with States.

**Highlights:**

- Updated for the latest Power BI platform to stay compatible with current and upcoming features
- Added a more advanced Color Rules editor for richer conditional formatting
- Unlocked all features with no license required
- Simplified the experience with cleaner options and easier setup
- Improved stability and performance with multiple fixes
- Renamed the previous generation to **Card with States Legacy**; it remains available for older reports but will no longer receive updates
