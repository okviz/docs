**Default value:** On

This option allows you to enable or disable the display of values in the tooltip.

## Display Unit

**Default value:** None

Specify the unit of measurement to display in the tooltip.

## Decimal Places

**Default value:** 0

Specify the number of decimal places to display in the tooltip.
