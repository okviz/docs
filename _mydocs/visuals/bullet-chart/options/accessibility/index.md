This section allows you to enable/disable some accessibility features.

## Data Label Tooltips

**Default value:** Off

When enabled, this option allows screen readers to read data labels on the bars while hovering them. This changes the default tooltip behavior for data labels.

## Arrow Keys Navigation

**Default value:** On

When enabled, you can move the keyboard focus between visual elements of the same type by using the arrow keys.
