**Default value:** Current Power BI locale

This option allows you to set the culture code used to format date values shown in the **Category** axis.

Use a culture code in the format `languageCode-countryCode`, for example `en-US`, `it-IT`, or `fr-FR`.

This option is relevant when the field connected to **Category** contains date values.

It does not affect numeric formatting. To control thousands separators and decimal separators, use [Number Formatting](number-formatting.md).
