This field allows you to bind one or more measures that can be referenced by the advanced color rules of Bullet Chart.

Values connected to **Color Rules** are not rendered directly in the visual. They are exposed to the rule editor so that you can create dynamic conditions based on additional measures.

Read more about [Conditional Colors](../options/conditional-color/index.md) and [OKVIZ Color Rules](../../features/color-rules.md).
