**Default value:** Top

This option allows the position of the legend within the visual area to be changed. It provides the following positioning options:

- **Top**
- **Bottom**
- **Right**
- **Left**
- **Top Center**
- **Bottom Center**
- **Right Center**
- **Left Center**
