## Hide Editor Button

**Default value:** Off

If enabled, this option allow to hide the advanced editor button.

## Pin Editor Button

**Default value:** Off

This option is only visible if the **Hide Editor Button** option is set to ***Off***. It allows the advanced editor button to be always visible.
